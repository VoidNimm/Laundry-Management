📌 STEP 1: DAPATKAN API KEY (2 menit)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Buka browser: https://makersuite.google.com/app/apikey
2. Login dengan Google Account
3. Click tombol "Create API Key" atau "Get API Key"
4. Pilih "Create API key in new project" (jika diminta)
5. Copy API Key (format: AIzaSy...)

📌 STEP 2: TAMBAHKAN KE FILE .ENV (1 menit)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Buka file .env di root project Anda
2. Tambahkan baris baru:

   GOOGLE_GEMINI_API_KEY="AIzaSy.............."
                          ↑ paste API key Anda disini

3. Save file

📌 STEP 3: RESTART SERVER (1 menit)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Stop server (Ctrl + C di terminal)
2. Start ulang:

   npm run dev

3. Buka browser: http://localhost:3000

✅ SELESAI! 
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ TROUBLESHOOTING:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
❌ Widget tidak muncul?
   → Check console browser (F12)
   → Pastikan sudah restart server
   
❌ Error "API Key invalid"?
   → Check .env file (tidak ada spasi ekstra)
   → Format: GOOGLE_GEMINI_API_KEY="AIzaSy..."
   → Copy ulang API key dari Google

❌ AI tidak menjawab?
   → Check internet connection
   → Check API key quota (1500 req/day free)
   → Lihat error di console browser
